# Shopping List App

Android-додаток для ведення списку покупок, розроблений на Kotlin + Jetpack Compose + Room.

**Студент:** Місько Руслан | ФІТ 1-3Мз

![alt text](image-2.png)
---

## Функціонал

- Список покупок з вертикальним скролом (LazyColumn)
- Додавання товарів через TextField + кнопку «Додати»
- Відмічання товарів як куплених (чекбокс + клік по картці)
- Закреслення тексту для куплених товарів
- Видалення товарів кнопкою 🗑
- Лічильник «Куплено: X / Y» над списком
- Збереження даних у локальній БД Room (SQLite)
- Дані та стани відновлюються після перезапуску додатку

## Стек

- Kotlin + Jetpack Compose (декларативний UI)
- Room 2.7.1 (локальна база даних)
- MVVM (AndroidViewModel + StateFlow)
- KSP (обробка анотацій Room)

## Власні доповнення

### 1. 🗑 Видалення товарів
Кнопка видалення на кожній картці товару.

### 2. Лічильник куплених товарів
Рядок «Куплено: X / Y» відображає прогрес покупок у реальному часі.

### 3. Анімація появи елементів
Нові товари з'являються з анімацією `fadeIn() + slideInVertically()`.

### 4. Свайп для видалення
Свайп картки вліво відкриває червоний фон з іконкою 🗑 та видаляє товар — як у нативних додатках.

## Труднощі

| Проблема | Вирішення |
|----------|-----------|
| `kotlin-kapt` несумісний з новою версією Android Studio | Замінено на KSP (`com.google.devtools.ksp`) |
| Room 2.6.1 + KSP → `unexpected jvm signature V` | Оновлено Room до версії 2.7.1 |
| Іконки `Icons.Default.*` не знаходились | Додано залежність `material-icons-extended` |
| KSP генерував sources через kotlin.sourceSets | Додано `android.disallowKotlinSourceSets=false` у `gradle.properties` |
